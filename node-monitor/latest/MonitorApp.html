Class: MonitorApp - extends YUI Widget

MonitorApp is a YUI Widget class representing the monitor application UI.

The widget is usually placed within the top element of a blank page.

Constructor

MonitorApp ( config )
Parameters:
config <Object> Configuration object (see Configuration attributes)

Properties

_body - protected Node

Reference to the Node instance containing everything below the header.

_browser - protected Node

Reference to the Node instance that will house the console messages.

_desktop - protected Node

Reference to the Node instance containing the desktop - the right portion of the application that contains the tabs for switching among monitor screens.

_evtCat - protected string

Category to prefix all event subscriptions to allow for ease of detach during destroy.

_footer - protected Node

Reference to the Node instance that contains the page footer

_header - protected Node

Reference to the Node instance containing the header contents.

_storage - protected object

The object stored into local storage for this _storageId.

_storageId - protected string

Persistent ID for this instance, used as the key to localStorage for persistent attributes. Set during construction based on the id attribute.

MonitorApp.APP_TEMPLATE - static String

HTML template for the application.

MonitorApp.NAME - static String

The identity of the widget.
