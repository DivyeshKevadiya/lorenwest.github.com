Class: MonitorEvent

Monitor event model These objects are created by probes, and sent when they emit the monitor event. They represent a single event that happened in time. These objects pass through filters and between processes.

Constructor

MonitorEvent ( name , pId , data )
Parameters:
name <String> - Name of the event being monitored
pId <Integer> - The registered ID of the probe that generated the event
data <Object> - Additional data to monitor (optional)
