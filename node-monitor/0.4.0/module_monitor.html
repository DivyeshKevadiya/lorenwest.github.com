Module: monitor

To get started, place the following in your application bootstrap: 

var Monitor = require('monitor');
Monitor.start('My Application');

This starts the default monitors and allows the UI to attach to this process.

To start the UI in this process:

Monitor.startUI();

To start the UI in a separate process:

npm monitor start

Then point your browser to port 4200

This module contains the following classes:

  • Monitor
  • MonitorEvent
  • Probe
  • Process
